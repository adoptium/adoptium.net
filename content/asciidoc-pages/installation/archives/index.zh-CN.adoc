= 归档安装
:page-authors: gdams, tellison, NickJavaDev88
:toc:
:icons: font

归档文件提供了一种无需包管理器即可安装 Eclipse Temurin 的简便方法。从
link:/temurin/releases[发布页面] 下载适合您平台的归档文件，然后按照以下说明操作。

[NOTE]
====
对于大多数用户，我们建议使用 link:/installation/linux[包管理器] 或
link:/installation/github-actions[CI/CD 集成] 而不是手动归档安装。归档文件最适合
自定义环境或不支持包管理器的平台。
====

== 验证您的下载

所有 Eclipse Temurin 归档文件都包含 SHA-256 校验和文件。我们强烈建议在安装前验证
您的下载。

=== Linux / macOS / AIX / Solaris

[source, bash]
----
# 在下载的二进制文件所在的目录中运行此命令
wget -O- -q -T 1 -t 1 <校验和文件的URL> | sha256sum -c
----

=== Windows

[source, powershell]
----
certutil -hashfile <下载的文件>.zip SHA256
----

TIP: 校验和文件可在 link:/temurin/releases[发布页面] 上每个二进制文件旁边获取。
您也可以通过 link:https://api.adoptium.net/q/swagger-ui/[Adoptium API] 验证下载。

== Linux

[source, bash]
----
# 解压归档文件
tar xzf <openjdk二进制文件>.tar.gz

# 添加到 PATH
export PATH=$PWD/<解压目录>/bin:$PATH

# 验证
java -version
----

要使此更改永久生效，请将 `export PATH` 行添加到您的 `~/.bashrc`、`~/.zshrc` 或
等效的 shell 配置文件中。

== macOS

[source, bash]
----
# 解压归档文件
tar xzf <openjdk二进制文件>.tar.gz

# 添加到 PATH（注意 Contents/Home 子目录）
export PATH=$PWD/<解压目录>/Contents/Home/bin:$PATH

# 验证
java -version
----

NOTE: macOS 归档文件包含 `Contents/Home` 目录结构，这反映了标准的 macOS JDK 布局。

== Windows

[source, powershell]
----
# 解压归档文件
Expand-Archive -Path .\<下载的文件>.zip -DestinationPath .

# 添加到 PATH（命令提示符）
set PATH=%cd%\<解压目录>\bin;%PATH%

# 验证
java -version
----

TIP: 要在 Windows 上永久设置 `JAVA_HOME` 和更新 `PATH`，请使用
*系统属性 > 环境变量*，或运行：
`setx JAVA_HOME "C:\path\to\<解压目录>"` 和
`setx PATH "%JAVA_HOME%\bin;%PATH%"`。

== AIX

[WARNING]
====
能够在 AIX 7.1 上运行的最后一批 Eclipse Temurin 版本是 8u362、11.0.18 和 17.0.8。
更高版本需要 IBM XL C/C++ 运行时包：

* link:https://www.ibm.com/support/pages/ibm-xl-cc-runtime-aix-131[XLC 13.1] 用于 JDK 8
* link:https://www.ibm.com/support/pages/ibm-xl-cc-runtime-aix-161[XLC 16.1] 用于 JDK 11、17 和 21
* link:https://www.ibm.com/docs/en/openxl-c-and-cpp-aix/17.1.2?topic=reference-open-xl-cc-runtime-environment-filesets[OpenXL 17]（需要 AIX 7.2 TL5 SP3 或更高版本）
====

[source, bash]
----
# 解压归档文件（AIX tar 不支持 z 标志）
gunzip -c <openjdk二进制文件>.tar.gz | tar xf -

# 添加到 PATH
export PATH=$PWD/<解压目录>/bin:$PATH

# 验证
java -version
----
