== 概览

https://projects.eclipse.org/projects/adoptium.aqavit[AQAvit^] 是针对 Java SE 运行时及相关技术的质量评估项目。AQAvit 采用功能完整的 Java 运行时，并确保其具备适合生产使用的所有额外品质。这些质量标准包括良好的性能、卓越的安全性、弹性和耐用性，以及通过各种应用程序测试套件的能力。

== AQAvit 软件产品验证

Eclipse AQAvit 测试作为开源许可软件https://projects.eclipse.org/projects/adoptium.aqavit/developer[免费向公众开放^]。您可以根据软件许可条款对 AQAvit 软件进行 fork、复制或修改。

指定的 https://github.com/adoptium/aqa-tests/releases[AQAvit 质量验证套件 (QVS) 版本^]根据 https://www.eclipse.org/legal/eclipse-foundation-quality-verification-suite-license.php[Eclipse 基金会质量验证套件许可证 (QVSL)^]提供。如果软件产品完全符合并满足 QVSL 中描述的 AQAvit QVS 的所有要求，则该软件产品将被视为通过了 AQAvit QVS “验证”。

无需成为 Adoptium 工作组成员即可生产通过 AQAvit 验证的软件产品。
